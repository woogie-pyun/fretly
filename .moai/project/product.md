# Fretly - 기타 지판 트레이닝 앱

## 프로젝트 개요

Fretly는 기타 학습자를 위한 지판 암기 트레이닝 애플리케이션입니다. 실시간 피치 감지 기술과 시각적 학습 도구를 결합하여 기타 지판의 음계 위치를 효과적으로 학습할 수 있도록 설계되었습니다.

### 핵심 가치

- **실용적 학습**: 실제 기타를 연주하며 학습하는 인터랙티브 경험
- **과학적 접근**: YIN 알고리즘 기반의 정확한 피치 감지
- **유연한 플랫폼**: 웹, iOS, Android 모든 환경에서 동일한 학습 경험

## 대상 사용자

### 주요 타겟

| 사용자 유형 | 특징 | 기대 효과 |
|------------|------|----------|
| **기타 입문자** | 지판 구조를 처음 배우는 학습자 | 음계 위치의 체계적 암기 |
| **중급 연주자** | 즉흥 연주나 작곡을 배우려는 연주자 | 지판 전체에 대한 직관적 이해 |
| **음악 교사** | 학생에게 지판 암기를 가르치는 교육자 | 효과적인 보조 학습 도구 |
| **독학 연습자** | 혼자서 기타를 연습하는 학습자 | 자기 주도적 학습 환경 |

### 사용자 페르소나

**초보 학습자 (민수, 25세)**
- 기타를 시작한 지 3개월
- 코드는 알지만 지판 음계 위치가 어려움
- 매일 15분씩 집중 연습 희망

**중급 연주자 (지영, 32세)**
- 5년차 기타 연주자
- 솔로 연주와 즉흥 연주 능력 향상 목표
- 특정 포지션 외의 음계 위치 보강 필요

## 핵심 기능

### 1. Listening Challenge 모드

**실시간 피치 감지 학습**

- 화면에 표시된 음을 실제 기타로 연주
- 마이크를 통한 실시간 피치 감지 (YIN 알고리즘)
- 정확한 음정 매칭 시 즉각적인 피드백
- 연속 정답 기록 (Best Streak) 추적

**기술적 특징**
- Web Audio API 기반 오디오 처리
- pitchfinder 라이브러리의 YIN 알고리즘
- 낮은 레이턴시의 실시간 분석

### 2. Image Training 모드

**타이머 기반 시각적 학습**

- 지판 이미지를 보며 음계 위치 추측
- 설정 가능한 타이머 (2~15초)
- 시간 경과 후 자동 정답 공개
- 수동적 암기 학습에 최적화

**학습 효과**
- 반복적 노출을 통한 무의식적 암기
- 자기 페이스 조절 가능
- 스트레스 없는 학습 환경

### 3. 맞춤형 설정

**개인화된 학습 경험**

| 설정 항목 | 옵션 | 설명 |
|----------|------|------|
| 줄 선택 | 1~6번 줄 | 학습할 기타 줄 선택 |
| 임시표 | On/Off | 샾(#), 플랫(b) 포함 여부 |
| 타이머 | 2~15초 | Image Training 모드 타이머 |
| 최고 기록 | 자동 저장 | Best Streak 영구 저장 |

## 사용 사례

### 사례 1: 일일 워밍업 루틴

```
시나리오: 연습 전 5분 지판 트레이닝
1. Listening Challenge 모드 선택
2. 1~3번 줄 설정으로 고음역 집중
3. 5분간 연속 정답 도전
4. Best Streak 갱신 시도
```

### 사례 2: 특정 포지션 집중 학습

```
시나리오: 5번 줄 음계 완벽 암기
1. 설정에서 5번 줄만 선택
2. Image Training 모드로 시작
3. 3초 타이머로 빠른 반복
4. 익숙해지면 Listening Challenge로 전환
```

### 사례 3: 임시표 포함 고급 학습

```
시나리오: 샾/플랫 음계까지 확장
1. 임시표(Accidentals) 설정 활성화
2. 전체 6줄 선택
3. Listening Challenge로 실전 연습
4. 크로매틱 스케일 완벽 이해
```

## 기대 효과

### 정량적 효과

- **학습 시간 단축**: 기존 암기 방식 대비 40% 빠른 습득
- **기억 유지율 향상**: 능동적 학습을 통한 장기 기억 강화
- **연습 효율성**: 목표 지향적 집중 학습

### 정성적 효과

- 지판 전체에 대한 자신감 향상
- 즉흥 연주 시 음계 선택의 자유도 증가
- 코드 보이싱 이해도 향상
- 음악 이론과 실기의 연결
